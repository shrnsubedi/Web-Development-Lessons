Projects

Color Game Project Color Game project is a simple game made with vanilla HTML/CSS/JS where you have to guess the color of the provided RGB values. You get to try again if your guess in incorrect. If you get the correct guess, you can play again. If it's too hard or too simple try out new colors.
To-Do List To-Do List is a simple application to demonstrate DOM manipulation in javascript. You can add new to-dos to the list or remove old ones. Be wary that this app doesn't have persistence of data, so don't acutally rely on it to save your todos.
YelpCamp Main Project YelpCamp is a NodeJS application where you can add or view various campgrounds. You can view the campgrounds or add a new campground that you have found. You can also comment on other campgrounds with your experience. The site provides login functionality, using passport.js. It uses express.js for routing and MongoDB for storing persistent data.

Budgety Budgety is a application to keep track of your income and expenses. The application keep track of your individual income and expense source and dynamically generates the budget for the month.
Pig-game Pig-game is a simple game made in vanilla JS. Two players are needed to play the game. Each player gets turn to roll the dice. Each dice roll gets accumulated to the score. You can hold the score or continue to roll. If you hit a 1, your score is set to 0. Whoever crosses 100 wins. The game is made entirely with vanilla JS.
